# 任务中心（独立产品版）

现在是两个独立入口：

- `/q1`：题目 1（Bilibili 下载器）
- `/q2`：题目 2（MySQL + GLM 流水线）

主页 `/` 只做任务选择，不把两题混在一个页面里。

## 交互方式

- 页面主体是对话流（聊天风格）。
- 参数默认折叠在输入框下方（点击展开）。
- 题目 2 的日志和 `result.json` 放在同一个“运行详情”选项中，支持锚点快速跳转。
- 题目 2 在每次运行后会弹出结果提示对话框。

## 安装

```bash
pip install -r requirements.txt
```

## 启动

```bash
python app.py
```

浏览器打开：

- <http://127.0.0.1:7860>
